# Console Access Control System

This is a simple C# console application developed to practice core programming concepts.

The application displays a menu where the user can choose different options.
All user input is validated to ensure that only numeric values are accepted.

The project focuses on:
- Control flow
- Loops
- Switch statements
- Input validation
- Console interaction

## Features

- Menu-based navigation
- Input validation using int.TryParse
- Loop control with while
- Clear console flow for better user experience
- Basic use of console colors for feedback messages

## Technologies

- C#
- .NET Console Application

## Purpose

This project was created as part of my learning journey in C# and backend development.
It is intended to demonstrate logic, structure, and clean control flow for junior-level roles.

## How to run

1. Clone the repository
2. Open the project in your IDE
3. Run the application from the console
